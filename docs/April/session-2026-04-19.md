# Session Summary — 19 April 2026

## Heaven Chart ("Heaven is the Decision I Must Make")

The Heaven chart was rebuilt as a fully HTML-rendered diagram in the Reference tab. The layout uses a CSS grid approach with the following structure:

The **Decision Maker** sits at the centre top as a gold badge. Two rows of pill-shaped arrow labels radiate from the centre — red arrows pointing left toward the Ego column, green arrows pointing right toward the Holy Spirit column. The first row shows "Denial of Innocence" (left) and "Denial of denial of Innocence" (right). The second row shows "Differences are important" (left) and "Sameness is important" (right).

Below the Decision Maker block, two equal columns present the Ego path (left, red-tinted) and the Holy Spirit path (right, green-tinted). Each column contains cause/effect chains, SVG fan diagrams (fragment/project for Ego, generalise/extend for Holy Spirit), and key teaching points.

Zoom functionality was added using the same pattern as the Dawson chart — plus/minus/reset buttons with a CSS transform-based zoom that preserves scroll position. The zoom container has no max-height restriction so the full chart remains accessible when zoomed in.

The function `buildHeavenChart()` in `build_html.py` generates the HTML. The `heavenZoom()` function handles zoom state and scroll position preservation.

## Special Completion Card (Lesson 366)

A "Where Do I Go From Here?" card was added as lesson 366 in the data. This is based on the appendix content from the Lesson and Reading Guide (pages 256–258), which covers whether to repeat the Workbook, post-Workbook practice structure from the Manual for Teachers (M-16), and hints for continuing practice.

The card is added to the LESSONS array via `add_special_card.py`, which appends it to `/tmp/app_data.json` before the build runs. The `renderCard()` function detects `num === 366` and applies special styling — gold label ("✦ COMPLETION ✦"), warm gradient background, gold border, and hidden external links. The library view shows a gold ✦ badge instead of a number and a gold left border.

The `gotoLesson()` range was updated from 365 to 366, and the goto input max attribute was changed accordingly. The daily lesson calculation (modulo 365) was left unchanged so the app never auto-navigates to 366 — users reach it intentionally.
